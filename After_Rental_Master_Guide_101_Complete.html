⛽ 1. FUEL CHARGE DISPUTES
REASON CODES

After Rental → Invoice Dispute - General → Fuel Charges Dispute

WHAT TO DO
  1. Check Connexus FIRST (most reliable)
    • Operations → Vehicles → Search registration → Telemetry → Fuel
    • Filter by RA number, check fuel at checkout vs check-in
    • Export PDF for evidence
  2. Check D&C App - Instrument cluster photos (delivery & collection)
  3. Check Documents - RA + SOR showing fuel policy and levels
  4. Check Greenway F7 notes - Station notes about delivery fuel
  5. Calculate if Over/Under 60 Miles
    • UNDER 60 miles: Need hard documentation (RA + SOR)
    • OVER 60 miles: Fuel gauge reading acceptable (8/8 = full)
WHEN TO SEND RFI
DO NOT RFI if: Connexus available OR D&C photos clear OR under 60 miles with no docs (just refund)
SEND RFI when: Customer claims fuel missing at delivery AND no D&C delivery photos available
To: Pickup station | SLA: 2 working days
WHEN TO MAINTAIN
✅ MAINTAIN when:
  • Connexus shows fuel missing at return
  • Over 60 miles + D&C/SOR shows not full (gauge below 8/8)
  • Customer admits not refueling (even if disputes amount)
  • Under 60 miles BUT signed RA + SOR + D&C all confirm not full
Documents to attach: Signed RA, Connexus PDF, D&C photos, SOR, T&C link
WHEN TO REFUND
❌ REFUND when:
  • Connexus shows returned full OR matches pickup level
  • Customer provides dated/time-stamped fuel receipt before return
  • Under 60 miles + no documentation after 5 working days (UK)
  • Delivery fuel not customer's responsibility (D&C/F7 shows missing at delivery)
  • Charged MORE than delivery fuel (e.g., delivered 7/8, returned 7/8 but charged)
  • No RFI response (UK: 2 days + escalation)
  • Price per litre doesn't match RA (refund difference via allowance)
CLOSING
If Maintaining If Refunding
Outcome: Charges Upheld - Connexus Justified
Root Cause: Customer		 Outcome: Refund OR Refund - Lack of Documented Evidence
Root Cause: Return Station OR Pickup Station (if delivery fuel)
🛡️ 2. COVERAGE CHARGES (CDW/THW/PROTECTION)
REASON CODES

Charges and Identification → Coverage → Coverage Charge Dispute
OR if insistence: Pickup and Return Time → Pickup Experience → Aggressive Sales / Insistence

WHAT TO DO
  1. Check RA + AOP Form - BOTH required for optional coverages
    • AOP must be signed, 100% correct, amounts EXACTLY match invoice
    • ANY error = refund (even 1p difference)
  2. Check Greenway (6,1) - Coverage on original booking?
  3. US/Canadian customers - Check if card coverage proof provided (CDW/TW only)
  4. Check country T&C - Mandatory coverage in some countries
WHEN TO SEND RFI
UK: DO NOT RFI - Wait 5 working days for docs, decide on day 6
International ONLY: If documents needed to maintain
SLA: 3 working days
WHEN TO MAINTAIN
✅ MAINTAIN when:
  • RA + AOP both signed, 100% correct, amounts exactly match invoice
  • On original booking + signed documentation
  • Mandatory coverage per country T&C (can maintain even without AOP)
  • International: Country provides signed docs OR RFI confirms
Documents to attach: Signed RA, Signed AOP, T&C link
WHEN TO REFUND
❌ REFUND when:
  • No AOP form after 5 working days (UK)
  • AOP incorrect - daily rate doesn't match, total doesn't match, fields blank, crossed out
  • Only RA present (no AOP) for optional coverages
  • Charged Medium AND Premium together (system error)
  • Not on booking AND not on paperwork
  • US/Canadian customer with valid card coverage document (refund CDW/TW only, NOT excess reduction)
  • International: No RFI response (3 days + escalation)
CLOSING
If Maintaining If Refunding
Outcome: Charges Upheld
Root Cause: Customer		 Outcome: Refund OR Refund - Lack of Documented Evidence
Root Cause: Pickup Station
📅 3. DAYS CHARGED / LATE RETURN / GRACE PERIOD
REASON CODES

After Rental → Invoice Dispute - Extra Services → Non-Compliance with Rental Agreement

WHAT TO DO
  1. Check Connexus Station Geofence (MOST RELIABLE)
    • Shows exact time vehicle entered return station
    • This is the true return time
  2. Check Greenway Screen 17 - Check-in time vs agreed return time
  3. Calculate Grace Period
    • Standard customers: 29 minutes
    • Star customers: 2 hours (check Customer Cluster field = "Star")
  4. Check SOR/D&C Collection - Time-stamped return documents
  5. Check F7 notes - Station authorization notes
  6. Check Vehicle Movement Screen - Next rental affected?
WHEN TO SEND RFI
SEND RFI when:
  • Customer claims station AGREED not to charge additional day
  • Customer adamant returned on time (station may have checked in late)
  • Early checkout - customer claims station agreed no charge
To: Return station (late) OR Pickup station (early) | SLA: 2 working days
WHEN TO MAINTAIN
✅ MAINTAIN when:
  • Connexus shows late return beyond grace period
  • SOR/Check-in docs show late return
  • Customer admits late (even if disputes amount) AND rate matches RA
  • Signed RA with new checkout time (early checkout agreed)
  • Standard customer claims 2-hour grace (incorrect - only 29 minutes unless Star)
Documents to attach: Connexus PDF, RA, SOR, T&C link
WHEN TO REFUND
❌ REFUND when:
  • Connexus shows returned WITHIN grace period
  • Customer provides dated/time-stamped evidence of on-time return
  • Star customer returned within 2-hour grace
  • Prepaid early return - rate INCREASED (cannot charge more than RA daily rate)
  • Station amended checkout time (not customer's request) - check Ctrl+E history
  • RFI confirms station authorized no charge
  • No RFI response + insufficient documentation
  • Vehicle Movement shows next rental checkout amended (indicates this customer on time)
  • Rate charged higher than RA states (refund difference)
SPECIAL: Prepaid early return with rate increase - calculate: Customer signed for £X/day, should pay only for actual days at that rate

Prepaid Early Return - Unused Days

NO REFUND for unused days per T&C

EXCEPTIONS (refund allowed):

  • Medical emergency with doctor's note
  • Mechanical breakdown (email Fleet Services - 14 day SLA)
  • Rate increased on early return (refund difference only)
CLOSING
If Maintaining If Refunding
Outcome: Charges Upheld OR Charges Upheld - Connexus Justified
Root Cause: Customer		 Outcome: Refund OR Refund - Lack of Documented Evidence
Root Cause: Return Station OR Pickup Station OR System Error (prepaid rate increase)
💳 4. DEPOSIT NOT RELEASED
REASON CODES

After Rental → Deposit → Deposit Not Released/Not Refunded After Rental
OR: After Rental → Deposit → Deposit Issue After Return

WHAT TO DO
  1. Check SixPay
    • Login: CustomerID: EuropcarEurope | Username: ECGB-CUSTSERV | Password: Summer25!
    • Search by first 6 + last 4 card digits OR RA number
    • Check Authorization tab - status: Active/Released/Expired
    • Check Transaction tab - actual charges taken
    • Reversal Receipt: IBatch → Credit Card Search → Reversal Receipts (UK from 21st Aug, full releases only)
  2. Check Greenway F12+W - Deposit amount, payment method
  3. Understand Timeline
    • Europcar releases within 24 hours of invoice
    • Bank processes: 3-10 working days (typical)
    • Maximum hold: 30 days (then expires)
WHEN TO SEND RFI
RARELY NEED RFI - SixPay shows all info
SEND RFI only when:
  • Manual authorization (not in SixPay) - station must release manually
  • Non-Greenway country - cannot confirm release status
WHEN TO MAINTAIN (Explain Only - No Refund)
✅ EXPLAIN PROCESS when:
  • Full Release - Within Banking Time: SixPay shows released, within last 10 days, send reversal receipt
  • Partial Release: Some charges taken, remaining released (bank tracks under original auth)
  • Within 30-Day Maximum: Will expire automatically
  • Manual Authorization: Contact station to request manual release
Outcome: Explanation | Root Cause: Customer
WHEN TO REFUND (Actual Refund Required)
❌ PROCESS REFUND when:
  • Deposit CHARGED not authorized (VPAY/Maestro cards) - actual money debited
  • Duplicate deposit charged (two separate deposits for same rental)
  • Wrong deposit amount charged (higher than reservation stated)
  • Authorization still active after 30+ days (should have expired)
Note: VPAY cards require BACS refund form (cannot use Greenway) - 10-15 working days
CLOSING
If Explaining (No Refund) If Refunding
Outcome: Explanation
Root Cause: Customer		 Outcome: Refund
Root Cause: Pickup Station OR System Error
👥 5. ADDITIONAL DRIVER
REASON CODES

Charges and Identification → Coverage → Additional Driver

WHAT TO DO
  1. Check Greenway Screen 17 - "Drivers and Waivers" section
    • Only ONE driver = likely incorrect charge
    • Multiple drivers = charge likely correct
  2. Check RA - Additional driver field completed and signed?
  3. Check if Privilege Member - Customer Cluster = "Star"? Check benefit tiers (some get free add driver)
  4. Check if Added Mid-Rental - F7 notes about when added (needs pro-rating)
WHEN TO SEND RFI
UK: DO NOT RFI for documentation - Wait 5 days
UK - ONLY for mid-rental timing: "When was additional driver added?" - to calculate pro-rata
International: If docs needed
SLA: 2 days (UK) / 3 days (Intl)
WHEN TO MAINTAIN
✅ MAINTAIN when:
  • Signed RA lists additional driver(s), daily rate stated, customer signed
  • Greenway Screen 17 shows 2+ drivers
  • On original reservation + signed documentation
  • Pro-rated correctly if added mid-rental (charged only for days entitled)
Documents to attach: Signed RA, Screen 17 screenshot, T&C link
WHEN TO REFUND
❌ REFUND when:
  • Only ONE driver on Greenway Screen 17
  • No RA after 5 working days (UK) OR RA section blank
  • Privilege member entitled to free additional driver
  • Duplicate charge - same driver listed twice (system error)
  • Not pro-rated (added mid-rental but charged for all days)
  • RA shows different rate than charged (adjust to RA amount)
  • International: No RFI response
Pro-rata calculation: Example - 7 days at £10/day = £70 charged, but driver added Day 3, entitled to 5 days only = £50, refund £20
CLOSING
If Maintaining If Refunding
Outcome: Charges Upheld
Root Cause: Customer		 Outcome: Refund
Root Cause: Pickup Station OR Changeover Station OR System Error (duplicate)

🚗 7. MILEAGE CHARGES
REASON CODES

After Rental → Invoice Dispute - Extra Services → Non-Compliance with Rental Agreement

KEY POINTS
  • Check Connexus Mileage Telemetry FIRST: Most reliable, shows exact odometer at start/end
  • Calculate Total: Mileage In - Mileage Out = Total Miles Driven
  • Check Allowance: Greenway (6,1) shows unlimited, per day, or total allowance
  • 27+ Day Rule: Rentals 27+ days = unlimited mileage after Day 27 per T&C
  • Watch for KM vs Miles: European vehicles use km (divide by 1.6 to get miles)
  • Runner Vehicles: High mileage between rentals = station transfers (not customer's responsibility)
MAINTAIN / REFUND
✅ MAINTAIN: Connexus confirms mileage | Signed RA + SOR show mileage, customer signed | D&C photos show odometer | Customer admits mileage, rate matches RA | 27+ days calculated correctly
❌ REFUND: Connexus shows LOWER mileage than charged | No Connexus + no docs after 5 days | Station amended mileage after return (check Vehicle Movement Screen) | Physically impossible mileage | KM charged as miles (÷1.6 to correct) | 27+ days - free mileage not applied | Price per mile higher than RA | Runner vehicle mileage included
CLOSING

Maintain: Outcome: Charges Upheld - Connexus Justified | Root Cause: Customer
Refund: Outcome: Refund OR Refund - Lack of Documented Evidence | Root Cause: Return Station OR Pickup Station

🔄 8. ONE WAY FEE
REASON CODES

After Rental → Invoice Dispute - Extra Services → One Way

KEY POINTS
  • Check Locations: Greenway (6,1) - Pickup vs Return branch codes
  • Technical One Way: SAME physical location, different codes (e.g., LHRS vs LHRP = Heathrow Standard vs Selection) = NOT CHARGEABLE
  • UK International Rule: UK one way NOT permitted outside UK - if returned abroad = repatriation charge (£500-£2,000+) MAINTAINABLE
SPECIAL AUTHORIZATION LIMITS: One Way has LOWER thresholds
Under £500: Process if fault | £500-£750: Station Manager | £750-£1,000: TM/RD | £1,000-£3,000: Operations Director
MAINTAIN / REFUND
✅ MAINTAIN: One way on original booking | Customer signed RA with one way fee | Returned to different location (not technical) | UK vehicle returned outside UK (repatriation cost per T&C)
❌ REFUND: Technical one way (same physical location) | Greenway shows SAME pickup and return | Amount doesn't match RA/booking | Not on booking AND not on paperwork | No RFI response
CLOSING

Maintain: Outcome: Charges Upheld | Root Cause: Customer
Refund: Outcome: Refund | Root Cause: Pickup/Return Station OR System Error

❌ REFUND: No signed RA/AOP for upgrade | Category on docs doesn't match charge | Customer claims forced (no evidence of acceptance) | Reservations promised specific vehicle

💧 12. REFUELLING SURCHARGE (Under 7 Litres Rule)
REASON CODES

After Rental → Invoice Dispute - General → Fuel Charges Dispute

KEY POINTS
  • The Rule: Refuelling surcharge (typically £25) ONLY applies if vehicle requires 7+ litres
  • Under 7 Litres: Customer pays ONLY for actual litres used (no surcharge)
  • Check Connexus: How many litres actually missing?
  • Common Issue: Customer used 6 litres but charged surcharge + litres = REFUND surcharge
MAINTAIN / REFUND
✅ MAINTAIN: Connexus shows 7+ litres missing | Customer charged: Litres + surcharge correctly
❌ REFUND SURCHARGE: Connexus shows UNDER 7 litres missing | Customer should pay litres only (no £25 surcharge) | Refund £25 surcharge via DEBCRED
CLOSING

Maintain: Outcome: Charges Upheld - Connexus Justified | Root Cause: Customer
Refund: Outcome: Refund | Root Cause: Return Station

❌ REFUND: No Connexus RPA alert (vehicle didn't cross border) | Customer provides evidence didn't travel abroad | Charged for country not visited | No toll usage evidence | Charged pack AND actual tolls (should be one or other)

📌 QUICK REFERENCE SUMMARY

🔍 Investigation Priority Order

  1. Connexus FIRST - Most reliable for fuel, mileage, location, cross-border
  2. Greenway F7 Notes - Check BEFORE any RFI
  3. Documentation - RA + AOP + SOR (wait 5 working days UK)
  4. D&C App - Photos prove condition/equipment/fuel
  5. RFI Only When Necessary - Don't RFI for docs (stations don't keep them)

⏰ Key SLAs

  • Case Resolution: 5 working days
  • UK RFI Response: 2 working days → TM → 1 day → outsourcesupport@europcar.com
  • International RFI: 3 working days → escalate per International Guidelines
  • UK Documentation: Wait 5 full working days, decide on day 6

💡 Golden Rules

  • 100% Correct or Refund: AOP forms must be PERFECT (any error = refund)
  • Connexus Overrides All: If Connexus contradicts other evidence, Connexus wins
  • Grace Periods: 29 minutes (standard) | 2 hours (Star customers UK)
  • Authorization Limits: Know yours - special limits for One Way, Days, Mileage, Category
  • Evidence-Based Only: Never guess - if no evidence, refund for lack of documentation

⚠️ Common Mistakes to Avoid

  • ❌ Not checking F7 notes before RFI
  • ❌ Sending RFI to stations for documents (they don't keep copies)
  • ❌ Making decisions before 5 working days (UK docs need time)
  • ❌ Assuming Connexus unavailable without checking
  • ❌ Maintaining charges when AOP has ANY error (must be 100% perfect)
  • ❌ Working Key Account cases (reallocate to UK-Key Account)
  • ❌ Processing refunds beyond your authorization limit
  • ❌ Closing cases that should be cancelled (awaiting info)
  • ❌ Not attaching evidence when maintaining charges
  • ❌ Using internal jargon in customer emails (RA, SOR, DEBCRED, etc.)

✅ Best Practices

  • ✅ Check Connexus FIRST for fuel, mileage, location queries
  • ✅ Always check F7 notes on invoice screen before ANY action
  • ✅ Export Connexus as PDF when it supports your decision
  • ✅ Attach ALL supporting documents when maintaining charges
  • ✅ Write detailed case comments explaining investigation and decision
  • ✅ Personalize ALL templates - never send unchanged
  • ✅ Use "Reply" button in Salesforce to maintain email threading
  • ✅ Check Customer Cluster field for Star/Privilege members
  • ✅ Calculate pro-rata refunds when customers only entitled to partial
  • ✅ Ask TL before closing if uncertain about decision

🔐 Special Rules to Remember

  • Star Customers: 2-hour grace period (not 29 minutes)
  • US/Canadian Customers: Can refund CDW/TW with card coverage proof (NOT excess reduction)
  • Prepaid Early Returns: No refund for unused days (except medical/mechanical with proof)
  • 27+ Day Rentals: Unlimited mileage after Day 27 per T&C
  • Under 60 Miles Fuel: Need hard documentation (RA + SOR), can't rely on gauge
  • Refuelling Surcharge: ONLY applies if 7+ litres missing (refund if under 7)
  • Technical One Way: Same physical location, different codes = NOT chargeable
  • UK International One Way: NOT permitted - repatriation charge applies
  • Mandatory Winter Equipment: Can maintain even without AOP (legally required)
  • Runner Vehicles: Station transfer mileage not customer's responsibility

✅ PRE-RESPONSE INVESTIGATION CHECKLIST

Before responding to ANY case, ensure you have:

📋 Basic Checks (ALL Cases)

  • Read customer email thoroughly - understand exact issue
  • Completed GDPR check - email matches account
  • Selected correct reason codes (all 3 levels)
  • Checked if Key Account (reallocate if yes)
  • Noted RA number, dates, amounts in question

🔍 System Investigation (Most Cases)

  • Checked Greenway F7 notes on invoice screen
  • Checked Greenway reservation (6,1) for original booking details
  • Checked Greenway Screen 17 for rental details and notes
  • Checked Salesforce "Retrieve All Docs" for uploaded documents
  • Checked KGS if 5+ working days elapsed (UK only)
  • Verified 5 working days have passed before deciding "no docs"

⛽ Fuel/Mileage/Location Cases

  • Checked Connexus telemetry FIRST (most reliable)
  • Exported Connexus PDF if data supports decision
  • Checked D&C App instrument cluster photos
  • Calculated correctly (under/over 60 miles for fuel)

📄 Coverage/Equipment Cases

  • Located signed Rental Agreement
  • Located signed AOP form (required for optional items)
  • Verified AOP is 100% correct (any error = refund)
  • Checked if item on original booking (Greenway 6,1)
  • Checked if mandatory (country T&C for winter equipment)

📅 Days/Grace Period Cases

  • Checked Connexus geofence for exact return time
  • Calculated grace period correctly (29 min OR 2 hours for Star)
  • Checked Customer Cluster field (Star status)
  • Checked Vehicle Movement Screen for next rental
  • Checked Ctrl+E for amendment history

💳 Deposit Cases

  • Checked SixPay authorization status
  • Checked SixPay transaction history
  • Located reversal receipt if full release (post-21 Aug UK)
  • Confirmed authorization vs transaction vs partial release

📊 Decision Making

  • Determined if RFI necessary (and to whom)
  • Reviewed Terms & Conditions relevant sections
  • Identified root cause (who is responsible)
  • Calculated refund amount if applicable (including VAT)
  • Confirmed decision within your authorization limit
  • Gathered all documents to attach if maintaining

✉️ Before Sending Response

  • Personalized template with case-specific details
  • Answered ALL customer questions
  • Attached supporting documents if maintaining charge
  • Checked spelling and grammar
  • Used customer's name correctly
  • No internal jargon (RA, SOR, DEBCRED, F7, etc.)
  • Professional but empathetic tone
  • Clear next steps provided

🔒 Case Closure

  • Selected correct Outcome
  • Selected correct Root Cause
  • Entered Refund Amount (if applicable)
  • Written detailed Case Comments
  • Processed refund via DEBCRED or form (if applicable)
  • Requested authorization if needed (before closing)
  • Saved case before closing
